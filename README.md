
Фикс для Debian 12 Bookworm, для установки с одной команды

Бан всех сетей/сканеров РКН с ежедневным обновлением листов от оригинального репозитория

## Установка одной командой

Выполните от **root**:

```bash
wget -qO- https://raw.githubusercontent.com/blablajka/AS_Network_List_for-debian/main/install.sh | bash
```
